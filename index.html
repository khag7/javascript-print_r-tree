Javascript print_r beautifier.

Converts scalar string output from PHP print_r function to collapsible DOM structure.

https://github.com/vearutop/javascript-print_r-tree

Usage

Processing previous element with disposable closure

<pre style="display:none">
MyClass Object
(
    ....
)
</pre><script src="js/print_r.disposable.min.js" type="text/javascript"></script>

MyClass Object
(
    [secret:MyClass:private] => secret data
    [arrayData:protected] => Array
        (
            [0] => 1
            [1] => 2
            [three] => 3
            [four] => Array
                (
                    [0] => 5
                    [1] => 6
                    [2] => 7
                    [3] => stdClass Object
                        (
                            [0] => 8
                            [1] => 9
                            [2] => 10
                            [3] => 11
                            [4] => 12
                        )

                )

        )

    [selfRecursion] => MyClass Object
 *RECURSION*
)
